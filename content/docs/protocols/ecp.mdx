# ECP Integration

ECP (Ethereum Comments Protocol) enables decentralized, censorship-resistant comments with optional gasless transactions.

## Overview

ECP provides:

- **Decentralized Comments** stored directly on-chain
- **Gasless Transactions** for easy UX
- **Moderation Tools**: Community-driven content filtering
- **Cross-platform**: Works across any ECP-enabled platform
