# ガスレス取引

[ECP](./protocols/ecp) は承認システムを備えており、コメントの投稿者が「自分に代わってコメントを投稿する権限」をアプリに委任できます。
これにより、ユーザーは一度だけ承認トランザクションに署名すれば、以後は毎回のコメント投稿で署名を求められません。
承認はいつでも取り消せます。

Paper は ECP のガスレス取引を実装しており、ユーザー操作の摩擦を取り除きます。

Paper にトランザクションのスポンサー（負担）を承認することで、Gas 料金を支払わずに Paper 上で投稿できます。

## 使い方

1. [設定](/settings) を開く
2. 「ガスレス投稿」までスクロール
3. 承認の有効期間を選択
4. 一度だけ承認トランザクションに署名（無料。支払うのはガス代のみ）

これで、あなたの投稿にかかるガス代は Paper が負担します。

詳しくは： [ECP 公式ドキュメント](https://docs.ethcomments.xyz/)
